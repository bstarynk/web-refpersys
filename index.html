RefPerSys, a free software symbolic artificial general intelligence project

RefPerSys - refpersys.org

An acronym for Reflexive Persistent System. This is an ambitious research project (still unfunded, in 2019 it is a hobby project; also very risky!), done on spare time with an open science mindset and ethics.

Early work in progress

We have in 2019 a lot of ideas, but not much code (on gitlab). Done by passionate developers on their spare time. Volunteers wanted; please send an email (introducing yourself) to basile@starynkevitch.net (near Paris, France, homepage here) for more.

design ideas

  • a mostly symbolic artificial intelligence long-term project, with ambitious Artificial General Intelligence goals.
  • for Linux distributions (such as Debian, Arch, Ubuntu, ....), on x86-64 desktops or laptops
  • free software, GPLv3+ licensed; some embryonic code (in C or C++, it should later become generated) available on gitlab.com/bstarynk/refpersys (for experimentation of ideas).
  • reflexive, persistent, multi-threaded, with a heavily metaprogramming and bootstrapping approach (all its code should become generated by the RefPerSys system, à la Quine), aiming to offer a declarative, knnowledge based language.
  • heavily inspired by the pionneering work of Jacques Pitrat (1934 - october 2019, France)

A draft document giving some design ideas is downloadable as refpersys-design.pdf. We are improving it in 2019.

The currently existing (in 2019, on gitlab) code base is a bit disappointing, even to the Refpersys team. But we have a lot of fun doing it. It is constantly improving, publicly available, still unreleased, α-stage.

dependencies

We list here some uncommon dependencies, and give a source tarball.

  • the omake build system is coded in Ocaml and is used in RefPerSys because it is a build automation system making decisions on file contents (and not on file modification time, like make does). Source tarball is omake-snapshot.tar.bz2 of md5sum 00eb0b094d107a1583218bf1347b7124
  • the Hjson-CPP library is a C++ library for Hjson, a compact derivative of Json format. Source tarball is hjson-cpp-1.3.tar.bz2 of md5sum 00eb0b094d107a1583218bf1347b7124
